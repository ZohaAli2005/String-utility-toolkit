# String-utility-toolkit
✨ String Utility Toolkit – Turning Text Into Magic! ✨

Just wrapped up a fun little project where CSS, HTML, and JavaScript came together to build a mini text transformation app. 

🔹 Clean UI with CSS – a centered container, soft shadows, rounded corners, and interactive buttons that feel smooth and modern.
🔹 Interactive JS Functions – one click and your text can:

🔠 Convert to Uppercase

🔡 Switch to Lowercase

🔄 Reverse every character

✨ Capitalize each word beautifully

 🔹 Responsive Layout – simple, minimal, and user-friendly.



This project reminded me how powerful small utilities can be when paired with clean design and functional logic. Sometimes, it’s the little tools that make the biggest impact! 💡
